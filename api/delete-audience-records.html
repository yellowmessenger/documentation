Delete Audience Records

Delete one or more audience records by their IDs.

Query Parameters
bot string — REQUIRED

The bot ID specific to your environment.

Request Body — REQUIRED
idArray string[]

Array of audience record IDs to delete.

audienceId string

Audience ID linked to the records being deleted.

Responses
200

Records deleted successfully.

Schema — OPTIONAL
message string — OPTIONAL
deletedRecords string[] — OPTIONAL
400

Invalid request or missing parameters.

401

Unauthorized request. Check your API key.
