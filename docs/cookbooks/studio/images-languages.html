How to display images based on different languages?

  1. Click the Language button on the left tile and add the preferred languages using +Add language.
  1. Create a global variable for language.
  1. Add a variable node, in Name include the global variable and in Value enter one of the preferred language codes.

  1. Add Set Language node and add the global variable here.

  1. Add an image node and add an image to it. This image will be mapped to the language code you entered in step 3.

  1. Click the Language button on the left tile again, and select the language you want to update. Navigate to the image node, and change the current image with the corresponding image for the selected language. Repeat this step for all the languages.

  1. Set a language code in the variable node to display the respective image in the bot
